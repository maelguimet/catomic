# Catomic v0.1 Roadmap Acceptance

Last verified: 2026-07-16.

The defined Phase 0 through Phase 8 roadmap is complete. Phase-specific evidence
is retained in the acceptance records for
[Phase 2](phase-2-acceptance.md), [Phase 3](phase-3-acceptance.md),
[Phase 4](phase-4-acceptance.md), [Phase 5](phase-5-acceptance.md),

tests in the current gate.

## Shipped Contract

- A Linux-first modeless terminal editor with Unicode-safe PieceTable editing,
  incremental line indexing, selection, search, undo/redo, atomic save, external
  change protection, multiple buffers, and editable paged large files.
- Viewport-only light syntax, Markdown preview, line/whitespace indicators,
  bounded Plain completion, and explicitly gated Project discovery, linting,
  diagnostics, and cached path completion.
- Explicit, bounded, confirmed current-file and repository-brokered LLM flows;
  validated single-file output, read-only preview, drift refusal, and one-step
  undo. No client or network call exists before explicit confirmation.
- Typed TOML, language settings, keybinding overrides, trusted but guarded named
  commands, ordered lifecycle hooks, and preview-before-edit semantics.
- Restrained cat polish plus opt-in private `.catnap` recovery that never replaces
  source automatically.

  all 10 real binary PTY smokes passed.
- `cargo test --all-targets -- --ignored --test-threads=1 --nocapture`: all 13

- `cargo package --allow-dirty --no-verify --list`: package contents resolved.

No live-model or live-endpoint command was run.

## Deliberate Non-Goals

The completed v0.1 contract does not include a full LSP client, tree-sitter,
embedded scripting/plugin ABI, multi-file LLM apply, `:feralmeow`, split views,
non-Linux first-class support, or a heavy theming/widget framework. These are
explicit later/non-goal items, not incomplete Phase 0–8 exit criteria.

## Result

Catomic v0.1 is implementation-complete against its roadmap. Publishing tags,
prebuilt artifacts, or registry releases remains a separate release operation
because it changes external state.
